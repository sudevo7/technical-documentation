Introduction to JavaScript

JavaScript is a programming language that is commonly used to create interactive effects within web browsers.

JavaScript allows you to implement complex features on web pages.

Every time a web page does more than just sit there, displaying static information, JavaScript is probably involved.

It allows developers to create dynamic and interactive websites.

JavaScript is one of the core technologies of the web, along with HTML and CSS.

Variables

Variables are containers for storing data values.

In JavaScript, variables are declared using the let or const keywords.

Here's an example:

let x = 5;

The const keyword is used to declare a constant variable, which cannot be changed.

  • let: Used for variables that can be reassigned.
  • const: Used for variables that cannot be reassigned.
Data Types

JavaScript provides different data types to hold different types of values.

  • String: Represents textual data.
  • Number: Represents numeric values.
  • Boolean: Represents true or false values.
  • Object: Represents structured data.

Here's how you declare a string:

let name = "John";
Operators

Operators are used to perform operations on variables and values.

Here are some basic operators in JavaScript:

  • + (Addition)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)

Example of using an operator:

let sum = 5 + 10;
Functions

A function is a block of code designed to perform a particular task.

To define a function in JavaScript, use the function keyword:

function greet() {
  console.log("Hello World");
}

Functions can take parameters and return values:

function add(a, b) {
  return a + b;
}

Functions are invoked by calling their name followed by parentheses.
